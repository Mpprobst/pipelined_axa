This directory contains two test programs: 
1. JumpTests.txt - assembly code used to test functionality of all branch and jump instructions
2. Tests.txt - assembly code used to test functionality of all other instructions

Other files include:
1. vmem0.txt - register file. Our tests initialize all registers to 0.
2. vmem1.txt - data memory segment.
3. JumpTests_vmem2.txt - is JumpTests as an assembled program memory
4. Tests_vmem2.txt - is Tests as an assembled program memory
